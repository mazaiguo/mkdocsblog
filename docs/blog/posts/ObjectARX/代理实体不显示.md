# 自定义实体不显示

## 让代理实体不显示

### 重载以下代码

```cpp
virtual void saveAs(
    AcGiWorldDraw* mode,
    AcDb::SaveType st
);
```

### 实现以下代码

```cpp
void CPipeEntity::saveAs(AcGiWorldDraw* mode, AcDb::SaveType st)
{
    switch (mode->regenType())
    {
    case kAcGiSaveWorldDrawForProxy:
        //如果是代理实体，什么都不显示
        break;
    default:
        midWorldDraw(mode);
        break;
    }
}
```
